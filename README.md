# KoopmanRL Landing Page

## About

This is the landing page for _Koopman-Assisted Reinforcement Learning_. For changes to the landing page, please submit a PR.

## Further Steps

* [KoopmanRL](github.com/dynamicslab/KoopmanRL): A reusable package to build on top of Koopman-Assisted Reinforcement Learning
* [HydroGym](github.com/dynamicslab/HydroGym): A full-fledged reinforcement learning framework for science offering an integration with Koopman-Assisted Reinforcement Learning for ready benchmarking on flow control problems.
